# Предсказание массы зелёных черепах (*Chelonia mydas*)

## Описание проекта
Проект посвящён построению модели машинного обучения для предсказания массы черепах по морфометрическим признакам. Исследованы линейные модели (LinearRegression, Lasso, Ridge, SGDRegressor) с различными способами масштабирования данных.

## Датасет
Исходные данные содержат информацию о 6 видах черепах. Для моделирования выбран вид *Chelonia mydas* (2829 записей). Признаки: длина и ширина панциря, головы, ласт, количество колец (возрастной признак).

## Используемые библиотеки
- pandas, numpy – обработка данных
- matplotlib, seaborn – визуализация
- scikit-learn – модели машинного обучения

## Результаты
Лучшая модель – Ridge-регрессия с alpha=10 на нормализованных признаках. Достигнутые метрики на тестовой выборке:
- MAE = 4.75 кг
- RMSE = 6.32 кг
- R² = 0.9666
- MAPE = 4.96%

Модель близка к целевым показателям и может быть рекомендована к использованию с осторожностью.
