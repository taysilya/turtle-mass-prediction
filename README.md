## 📌 Описание проекта

Центр реабилитации и исследования морских черепах «Тортилла» занимается спасением и изучением зелёных морских черепах (*Chelonia mydas*). Для мониторинга здоровья животных важно знать их массу, однако традиционное взвешивание требует контакта и может вызывать стресс у животных.

В рамках проекта разработана модель линейной регрессии, которая по габаритным размерам черепахи (длина и ширина панциря, размеры головы и ласт, количество колец на щитках панциря), получаемым с помощью подсистемы компьютерного зрения **TurtleCV**, позволяет бесконтактно предсказывать массу особи.

## 🎯 Цели и задачи

1. Провести исследовательский анализ данных (EDA) о черепахах.
2. Очистить и подготовить данные для машинного обучения.
3. Обучить несколько моделей линейной регрессии.
4. Выбрать лучшую модель по метрикам качества.
5. Оценить важность признаков.
6. Разработать финальную функцию прогнозирования для интеграции с прибором TurtleCV.

**Критерии успеха модели:**
- **R² ≥ 0.97** на валидационной и тестовой выборках.
- **MAE ≤ 5 кг** для взрослых особей (масса 50–150 кг).
- Минимизация **MAPE** для работы с особями разного возраста.

## 📊 Данные

Исходный датасет (`turtles.csv`) содержит 8861 запись и 20 признаков, среди которых:
- `id` – идентификатор измерения;
- `binomial_name` – научное название вида;
- `registration number` – регистрационный номер;
- `shell_length`, `shell_width` – длина и ширина панциря (мм);
- `head_length`, `head_width` – длина и ширина головы (мм);
- `flipper_length_n`, `flipper_width_n` – длина и ширина четырёх ласт (мм);
- `circle_count` – количество колец роста;
- `measure_count` – количество измерений для усреднения;
- `shell_crack` – наличие трещин панциря;
- `timestamp` – время внесения данных;
- `weight` – масса черепахи (кг) – **целевая переменная**.

После фильтрации по виду *Chelonia mydas* осталось 2829 записей.

## 🔍 Ход работы
1. EDA:

Изучено распределение видов, отобраны только Chelonia mydas.

Проанализированы пропуски, дубликаты, выбросы.

Построены гистограммы, ящики с усами, корреляционная матрица (phik).

Обнаружены выбросы в shell_length и shell_width, скорректированные делением на 10.

2. Предобработка:

Удалены строки с пропущенной weight.

Исключены неинформативные признаки (id, binomial_name, registration number, measure_count, shell_crack, timestamp).

Удалены явные дубликаты.

Проведено масштабирование признаков.

3. Моделирование:

Данные разделены на train / validation / test.

Обучены: LinearRegression, Lasso, Ridge, SGDRegressor.

Сравнение моделей по метрикам MAE, MSE, R², MAPE.

Выбор лучшей модели – линейная регрессия с регуляризацией (Ridge) показала наилучшие результаты.

4. Интерпретация:

Оценена важность признаков (коэффициенты модели).

Наиболее значимыми оказались размеры панциря и количество колец.

5. Функция прогнозирования:

Реализована функция predict_weight(), принимающая измерения и возвращающая предсказанную массу.

##📈 Результаты
Итоговая модель достигла R² = 0.98 на тестовой выборке.

MAE для взрослых особей составила менее 3.5 кг.

Модель готова к интеграции в систему TurtleCV.
